# DWD.ejs

Portfolio and showcase website for Dakermanji Web Development.

## Stack

- Node.js
- Express
- MySQL
- EJS
- Bootstrap
- Helmet
- Winston
- Morgan
- Sentry

## Development

Install dependencies:

npm install

Run development server:

npm run dev

## Environment

Create a `.env` file based on `.env.example`.
